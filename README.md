### Currently

Learning by doing. Solving crackmes, dissecting binaries, documenting the process.

- **Static:** IDA Pro, Ghidra — decompilation, graph analysis, reconstructing logic from obfuscated code
- **Dynamic:** x64dbg — breakpoint chains, register patching, memory dumps at critical transitions
- **Subjects:** PE format, Windows internals, anti-debug techniques, position-independent shellcode
- **Languages:** x86/x64 Assembly (reading), C (basics), Python (automation scripts)
